

## 유용한 명령어 


- `;` 출력값 표시 차단, 입력시 화면에 출력 없이 실행 
- 긴 명령문 입력 마침표 3개  `'''`  vs. python `\`
 
 
## 프로그래밍과 스크립트

- 생성 : edit {파일명} `edit mysphere`
- 주석 : `$`
- 실행 : 파일명
